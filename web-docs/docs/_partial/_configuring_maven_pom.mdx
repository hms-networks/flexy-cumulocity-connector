_File Location: `pom.xml`_

The Maven POM file is used to configure the project's dependencies, build process, and
other project settings. This includes the project's name, version, description, and main class,
which must be configured before the project can be built.

### Step 1

Open the Maven POM file located at `pom.xml` in your favorite text editor.

### Step 2

Update the values of the following fields within the Maven POM
File: `name`, `version`, `groupId`, `artifactId`, `project.main.class`.

<img src={PomUpdateProjectInfo} alt="Maven pom file fields highlighted" />

### Step 3

<>After saving the changes to the Maven POM file, the project can be built, and you may continue
setting up and using the {ScDocusaurusConfig.title}.</><br/><br/>
