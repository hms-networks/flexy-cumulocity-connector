| Logging Level | Description                                          |
|:-------------:|------------------------------------------------------|
|       0       | Logs disabled                                        |
|       1       | Include only critical logs                           |
|       2       | Includes serious logs as well as logs from level 1   |
|       3       | Includes warning logs as well as logs from level 1-2 |
|       4       | Includes info logs as well as logs from level 1-3    |
|       5       | Includes debug logs as well as logs from level 1-4   |
|       6       | Includes trace logs as well as logs from level 1-5   |
