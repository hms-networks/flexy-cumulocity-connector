## Description

<>The {ScDocusaurusConfig.title} is an application for synchronizing data from Ewon Flexy devices to
Cumulocity Cloud and Cumulocity Edge.</><br/><br/>

:::note

The connector uses the Cumulocity MQTT APIs for communication with Cumulocity Cloud and Cumulocity
Edge.

:::

## Getting Started

To get started, please refer to the [Quick Start Guide](03-QUICK_START_GUIDE.mdx). This guide
provides instructions for installing and configuring the connector, as well as, a brief overview of
the features and functionality provided by the connector.

<>Additional information about the {ScDocusaurusConfig.title} can be found on the various pages
available in the menu to
your left (above on mobile/small devices).</><br/>
